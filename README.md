# TO-DO List
A simple TO-DO list webpage using HTML, CSS and JS. All CRUD functions are working. The task are stored in the users system and on page reload, it will be reset.
## Operations Included
- Adding Task
- Deleting Taak
- Edit Task
- Mark Done
- Input Form
